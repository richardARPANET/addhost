# addhost.c

A simple program to set an entry in your [hosts](https://en.wikipedia.org/wiki/Hosts_(file)) file.

# Compile

```bash
clang -o addhost addhost.c
```

# Example Usage

Block The Man.

```bash
sudo ./addhost 127.0.0.1 analytics.google.com
```
